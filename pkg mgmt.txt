access control list
تستخدم لتحديد صلاحية لبعض المستخدمين دون الباقي علي ملف معين او dir كامله


عند انشاء يوزر فانه يتم انشاء جروب خاصة به بنفس الاسم

يمكن ان يكون اليوزر عضو في اكثر من مجموعه فرعيه

acl تكون مدعومة بشكل افتراضي في xfs

وفي حاله انها غير مدعومة: يجب عمل mount مع خيار تفعيل acl
mount -o acl
ويمكن التحقق من دعم النواة لل acl من خلال ملف config الموجود داخل ال boot


setfacl يستخدم هذا الامر لوضع مستخدم او جروب في acl
getfacl يستخدم لعرض acl الخاصة بملف معين

عند عمل acl ل dir يمكن جعل ال صلاحيات مورثة بحيث انه عند انشاء ملف جديد في ال dir يرث الصلاحيات ويمكن جعل الصلاحيات مقتصرة علي الملفات الحالية فقط

وعند وضع default permission يمكن ان تورث ولكنها تطبق مع الملفات التي تم انشائها بعد وضع الdef__

software management
---------------
code -> compile -> binary

package عبارة عن ال  binary + libs + backgrounds + icons
لولا وجود ال pkg mgmt كنا هنضطر ناخد كل ملفات البرنامج زي الخط والايقونات و binary ونحطها في مكانها وكذلك مشكله الاعتماديات

فكرة ال pkg mgmt هي وجود سيرفر مركزي خاص ب pkg mgmt زي yum وهيكون عنده بيانات metadata خاصه بالتطبيقات والاعتماديات بتاعتها ولما يتواصل معاه اي عميل بيسأل علي تطبيق هيبعتله التطبيق والاعتماديات بتاعتها
ويلزم معرفة عنوان السيرفر لنتواصل معه

يتم تحديث الميتاداتا عندك اذا تم تحديثها علي السيرفر او عندما يتم الاتصال بالسيرفر للمرة الاولي بعد تثبيت النظام

الاسطوانه اللتي تأتي مع ال os تحتوي علي ملف حاص بال rpm packages
وملف repodata يحتوي علي packages metadata

yum repolist يجلب لنا عدد الريبوز المتاحه للاستخدام
وتكون مسجله تحت /etc/yum.repos.d
وعند عرض احد هذه الملفات نجد عنوان الويب الخاصه بالحزم


عند التثبيت يبدأ تثبيت الاعتماديات اولا

yum info للحصول علي بيانات الحزمة


yum groupinstall يدعم تنزيل مجموعه حزم خاصه بوظيفة معينه مثل basic web server

yum provides بيجيب الحزمة الرئيسية اللي طلبت تنزيل الحزمة او انشأـ المللف ده

احيانا يحدث تحديث لميتاداتا الحزم ولا يتم التحديث علي الجهاز الخاص بنا ولإجبار تحديث الميتاداتا نستخدم امر yum clean

الافتراضي انه يمسح البيانات الخاصة بالحزم بعد التثبيت ونعدلها من ملف /etc/yum.conf
